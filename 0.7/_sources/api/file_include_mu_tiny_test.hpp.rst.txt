Primary test-definition header for mutiny. 


Definition (``include/mu/tiny/test.hpp``)
-----------------------------------------




Detailed Description
--------------------

Include this header in every C++ test file. It provides the macros needed to declare test groups and individual tests: :ref:`exhale_define_test_8hpp_1a109b3c7ca51a87da0abc0a229663d7fd`, :ref:`exhale_define_test_8h_1a387737a52be7d4ade8c76832ec677100`, :ref:`exhale_define_test_8h_1a235cffa951470b7e037566c4f8f201f6`, :ref:`exhale_define_test_8h_1acb496fa2cbcfaf0cb310ee4d264bf249`, and the C-interop wrappers. Assertion macros are provided by :ref:`file_include_mu_tiny_test_Shell.hpp`, which this header includes transitively.
