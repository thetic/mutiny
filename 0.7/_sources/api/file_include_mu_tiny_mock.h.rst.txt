Detailed Description
--------------------

Provides a vtable-style C API for mock interaction from pure C test code. Call :ref:`exhale_function_mock_8h_1a0ab307d5675f710eceb14b5d5567955e` to get the global mock scope, or :ref:`exhale_function_mock_8h_1ad749af26808dfba5e3404e9a2a3a62eb` for a named scope; then use the function pointers on the returned struct to set up expectations and report actual calls.
