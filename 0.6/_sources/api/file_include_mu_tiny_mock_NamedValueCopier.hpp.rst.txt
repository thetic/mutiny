Definition (``include/mu/tiny/mock/NamedValueCopier.hpp``)
----------------------------------------------------------




Detailed Description
--------------------

When a mock expectation is set with mu::tiny::mock::ExpectedCall::with_output_parameter_of_type_returning(), the framework needs to copy the configured value into the caller's output buffer. Implement :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1NamedValueCopier` for your type and register it with mu::tiny::mock::Support::install_copier().

See also :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1NamedValueComparator` for expected-parameter matching.
