Detailed Description
--------------------

Subclass :ref:`exhale_class_classmu_1_1tiny_1_1test_1_1Plugin` to intercept the test lifecycle, contribute a custom :ref:`exhale_class_classmu_1_1tiny_1_1test_1_1Output`, or handle extra command-line arguments. Plugins form a singly-linked list; install them with :ref:`exhale_class_classmu_1_1tiny_1_1test_1_1CommandLineRunner` before calling mu::tiny::test::CommandLineRunner::run_all_tests_main(). See also :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1SupportPlugin` and :ref:`exhale_class_classmu_1_1tiny_1_1test_1_1SetPointerPlugin` for built-in plugin implementations.
