Detailed Description
--------------------

This header is the primary source of assertion macros (:ref:`exhale_define_test_8h_1a3e1cfef60e774a81f30eaddf26a3a274`, :ref:`exhale_define_Shell_8hpp_1ad8fa79f5d491cc01af302832785e90fe`, :ref:`exhale_define_Shell_8hpp_1ade1dda09c948fee9ceb853bc6dd5f3cb`, :ref:`exhale_define_Shell_8hpp_1a7921a0c4f152a7879e78fe6fc2905590`, :ref:`exhale_define_Shell_8hpp_1a64bb32929e2c1f2ed5521a27e775327e`, etc.) and the :ref:`exhale_class_classmu_1_1tiny_1_1test_1_1Shell` class that backs them. It is included transitively by :ref:`file_include_mutiny_test.hpp`; you rarely need to include it directly.
