Detailed Description
--------------------

:ref:`exhale_class_classmu_1_1tiny_1_1test_1_1Failure` is the base class; each assertion macro creates an appropriate concrete subclass when the assertion does not hold. Failures are reported via mu::tiny::test::Shell::add_failure() and rendered by mu::tiny::test::Output::print_failure().
