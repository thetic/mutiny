-------------------------------------------------------------




Detailed Description
--------------------

When a mock expectation is set with mu::tiny::mock::ExpectedCall::with_parameter_of_type(), the framework needs to know how to compare the expected value against the actual value. Implement :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1NamedValueComparator` for your type and register it with mu::tiny::mock::Support::install_comparator().

Three concrete implementations are provided:

- :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1FunctionComparator` wraps plain function pointers

- :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1TypedMockComparator` uses ``T::operator==`` and :ref:`exhale_function_namespacemu_1_1tiny_1a7f4ee5d7ddb8b802bcaea8a49c8839e9`



See also :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1NamedValueCopier` for output-parameter support.
