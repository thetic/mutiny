Detailed Description
--------------------

Inside a mock function implementation, call mu::tiny::mock::Support::actual_call() and then chain mu::tiny::mock::ActualCall::with_parameter() for each argument before querying the return value. The framework matches the actual call against the registered expectations and fails the test if a mismatch is found. See also :ref:`exhale_class_classmu_1_1tiny_1_1mock_1_1ExpectedCall`.
