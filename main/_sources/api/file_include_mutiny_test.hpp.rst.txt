Definition (``include/mutiny/test.hpp``)
----------------------------------------




Detailed Description
--------------------

Include this header in every C++ test file. It provides the macros needed to declare test groups and individual tests: :ref:`exhale_define_test_8hpp_1a109b3c7ca51a87da0abc0a229663d7fd`, :ref:`exhale_define_test_8h_1a387737a52be7d4ade8c76832ec677100`, :ref:`exhale_define_test_8h_1a2bcb6de8c4e02b2f6f8a93de8bdf3f8f`, :ref:`exhale_define_test_8h_1a131c35b131f7ee5d66b668eb151a63fe`, and the C-interop wrappers. Assertion macros are provided by :ref:`file_include_mutiny_test_Shell.hpp`, which this header includes transitively.
