Detailed Description
--------------------

:ref:`exhale_class_classmu_1_1tiny_1_1test_1_1SetPointerPlugin` records the original address stored in a pointer variable before it is overwritten and restores it at the end of every test, without requiring explicit teardown code.

Use the :ref:`exhale_define_SetPointerPlugin_8hpp_1ab46a6c558e87a029cffe107b358a2691` macro (the preferred interface) or call mu::tiny::test::SetPointerPlugin::set_pointer() directly.
