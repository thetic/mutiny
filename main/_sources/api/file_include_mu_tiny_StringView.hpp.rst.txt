Detailed Description
--------------------

StringView stores a pointer and a length without allocating or copying. It is always the custom implementation regardless of :ref:`exhale_define_features_8hpp_1a2b5a2087ad8293d85d2612da40675a1b`, keeping the type trivially copyable and free of standard-library dependencies.
